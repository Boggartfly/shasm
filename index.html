Shasm

A simple shell assembler

View on GitHub Download .zip Download .tar.gz

Usage sh shasm.sh OR ./shasm.sh

shasm works on Linux and Mac OS X. Requires Xcode on Mac OS X. May work any other shell having as and ld installed on the system. This is Free Software as defined by Under GPL V2, further details within LICENSE file.
